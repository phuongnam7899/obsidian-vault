# Mode change
- Normal Mode: `ESC`
- Insert Mode:
	- Insert before: `i`
	- Insert after: `a`
	- Insert in new line: `o`
- Visual Mode (For marking text)
	- Start visual mode: `v`
# Cursor movement
- Single
	- Left: `h`
	- Right: `l`
	- Up: `k`
	- Down: `j`
- Word:
	- Start of next word: `w`
	- Next end of word: `e`
	- Backward to start: `b`
- Line:
	- Start of line: `^`
	- End of line: `$`
- File:
	- Go to local declaration: `gd`
- Workspace:
	- Go to global declaration: `gD`
# Edit
- Replace until end of word: `cw`
- Undo: `u`
- Redo: `Ctrl+r`
