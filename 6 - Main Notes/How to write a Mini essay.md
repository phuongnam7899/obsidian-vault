# Rules
- Only cover 1 distinct idea
- Keep them short (2-500 words)n
- Keep them organized
# Structure
- Intro
	- Goals:
		- Make the premise, clear
		- Spark curiosity
	- Styles:
		- A strong statement
		- Inspire emotion
		- Ask a question
- Body
	- Goal:
		- Explain, add details
	- Rule:
		- Make things connected
		- Break body into paragraphs
		- Tell stories
- Conclusion
	- Need to be short and easy to remenber
