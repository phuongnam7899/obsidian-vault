## List all container

```sh
docker ps
```

## Build image from Dockerfile
```sh
docker build -t tag-name .
```
- `-t`: tag - give the image a tag to run later
- `.`: the path to the `Dockerfile`

## List all images
```sh
docker images
```

## Run a container from image
```sh
docker run -d --name
```
